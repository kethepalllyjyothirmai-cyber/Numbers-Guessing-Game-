# Number Guessing Game 🎯

## Overview

This project is a Number Guessing Game developed in Python. The program generates a random number between 1 and 100, and the user attempts to guess the correct number. Based on the user's input, the program provides feedback indicating whether the guess is too high, too low, or correct.

## Features

* Random number generation
* User input handling
* Instant feedback on guesses
* Simple and interactive gameplay
* Beginner-friendly Python project

## Technologies Used

* Python 3
* Random Module

## How It Works

1. The program generates a random number between 1 and 100.
2. The user enters a guess.
3. The program compares the guess with the generated number.
4. It displays whether the guess is:

   * Correct
   * Too High
   * Too Low
5. The correct number is revealed if the guess is incorrect.

## Concepts Used

* Variables
* User Input and Output
* Conditional Statements (`if-elif-else`)
* Random Number Generation
* Type Conversion (`int`)
* Problem Solving

## Example

Input:

* Guess: 50

Output:

* Too Low! The correct number was 78.
